   Как скачать и установить?
   1) Python 3.8
  Для начала работы нужно скачать и установить Python 3.8 (https://www.python.org/downloads/release/python-380/)
 Можно использовать Windows x86 executable installer.exe (Предпоследний в списке)
При установке Обязательно ставим галочку напротив надписи Add Python 3.8 to PATH
Далее в списке выбора нужно выбрать чтобы PIP тоже установился

После этого жмем горячие клавиши WIN + R (windows) и пишем CMD
У нас открывается командная строка и пишем туда python --version
Если у нас появилась надпись Python 3.x.x (В данном случае 3.8.0) Значит мы сделали все правильно
   2) Запуск программы(если по двойному клику не работает)
  Для начала распакуйте архив в любую папку на диске C
 После распаковки открываем консоль(так же как и в прошлый раз)
Копируем путь до корневой директории с софтом
Пишем в консоль cd C:\путь\до\папки
После того как перешли в папку с софтом пишем одну из команд(в зависимости от операционной системы):
python3 main.py
python main.py
main.py

Софт запускается







   Инструкция по использованию:
   После запуска софта мы попадаем в меню:
1. Показать все заметки
2. Найти заметку по ключевому слову
3. Добавить новую заметку
4. Удалить заметку


Для того чтобы получить ответ нужно написать цифру нужной функции

1 - Покажет все ваши ранее записанные заметки(Заголовок, содержание и уникальный ID)
2 - Найдет заметку по ключевому слову. Важно что поиск по ключевому слову осуществляется по заголовку И по содержанию.
  Также он не чувствителен к регистру!
3 - Добавляет новую заметку, вводите Имя заметки -> Жмете Enter -> вводите содержание заметки -> Жмете Enter И готово
4 - Удаляет заметку, для этого нужен ID заметки(он отображается рядом с именем заметки)
